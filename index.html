  1. Basic Broiler Plate

    2. The broiler plate is the main structure for formulating any web page. It basically gives a skeletal structure to put things in order, consisting mainly of the head and the body.

    The Head contains the details of the page such as its title, the meta tags or if the HTML file have any sort of attached style sheets, etc. It's not included in the visible area of a running web page.

    Then comes the Body, which contains almost everything we see in a web page. It contains numerous tags and texts and designs required to build a webpage as per convinience.



  2. Heading Tags

    4. The order and unordered list tags are used for making ordered and unordered lists. Whereas in an unordered list the items are represented with bullets of differnt types. Here is an example of each of them respectively.

      This is an ordered list:

    2. 2nd Item
    3. 3rd Item

      This is an unordered list:
    • 1st Item
    • 2nd Item
    • 3rd Item
